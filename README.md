# Athaliana-processing-scripts

This project is a collection of scripts used to process helixer- and liftoff-annotated files of 458 well-sequenced _Arabidopsis thaliana_ accessions.
The project was supervised by Dr. Gal Ofir and Sicheng Xu, who helped me set goals and gave me hints on what to do next.

The goal of the project was to find a set of 510 embryonal-essential genes (also called EMB-genes, [DW Meinke, 2018](https://pubmed.ncbi.nlm.nih.gov/31334862/)) in all of the accessions. 
Files were aligned using MMseq2.

Some analysis was done in R, though most scripts are bash-based.
